# Desafio: Sistema de Numeração de Andares de Hotel

Este projeto foi desenvolvido como parte do curso de Programação e Desenvolvimento Web. O objetivo é aplicar conceitos de laços de repetição em Python, respeitando uma regra de negócio específica: pular o 13º andar por superstição do proprietário.

## 🚀 O que este código faz:
- Percorre os andares de 1 a 20 e imprime cada um deles.
- Utiliza a instrução continue para saltar o número 13.
- Apresenta os resultados em ordem crescente e decrescente.

## 🛠️ Tecnologias Utilizadas:
- *Linguagem:* Python 3
- *Ambiente:* Ubuntu Linux
- *Controle de Versão:* Git e GitHub

## 📝 Como executar:
Para rodar o projeto localmente, utilize o comando:
python3 hotel.py
